Nosense Word Generator

Use this generator to make some made-up words.

How to use: Use the following letters in the "Pattern" input box to generate some words that adhere the pattern.

V: Vocal letter

C: Consonant letter

You may customize vocal or consonant letters in "Advanced Generator"

Example: If you input "VCVCC" (without quotation) in the "Pattern" input box with default vocal and consonant letters, the possible outcomes may be "utowt", "ikawh", etc.
